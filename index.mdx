### **🔧 简单易用**

切换模型就像修改一个参数一样简单：

```python
# 使用 GPT-4
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello!"}]
)

# 切换到 Claude 3
response = openai.ChatCompletion.create(
    model="claude-3-opus-20240229",  # 只需修改模型名称
    messages=[{"role": "user", "content": "Hello!"}]
)
```
